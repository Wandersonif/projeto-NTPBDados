# Dashboard de Vendas Fictícias - NTPBDados

Este projeto consiste em um sistema de análise de vendas com gráficos e relatórios em tempo real, desenvolvido para a disciplina de Banco de Dados.

## 🚀 Tecnologias
- **Frontend:** React.js + Chart.js
- **Backend:** Node.js + Express
- **Banco de Dados:** PostgreSQL

## 📂 Estrutura do Projeto
- `/frontend`: Aplicação client-side em React.
- `/backend`: API server-side em Node.js.
- `/docs`: Documentação técnica e modelos de banco de dados.

## 🛠️ Padronização de Commits
Para manter a organização, utilizaremos o padrão de **Conventional Commits**:
- `feat:` Novas funcionalidades.
- `fix:` Correções de bugs.
- `docs:` Alterações na documentação.
- `style:` Formatação, falta de ponto e vírgula, etc (sem alteração de código).
- `refactor:` Alteração de código que não corrige bug nem adiciona funcionalidade.
- `test:` Adição de testes.
- `chore:` Atualização de tarefas de build, configurações de pacotes, etc.

Exemplo: `feat: adiciona gráfico de vendas por categoria`
